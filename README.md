# Password Strength Checker

This Python program checks whether a password is strong or weak.

The script evaluates:

- Password length
- Whether the password contains letters
- Whether the password contains numbers or symbols

Example:

Input:
mypassword

Output:
Medium password: add numbers or symbols

Skills demonstrated:

- Python conditionals
- Input validation
- Basic security logic
